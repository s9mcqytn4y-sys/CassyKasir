# Cassy Kasir

Cassy Kasir adalah aplikasi Point of Sales Android Native modern untuk UMKM.

## Posisi Produk
- ringan
- cepat
- modern
- mudah dipakai
- tepercaya saat jam sibuk

## Tujuan Proyek
Membangun aplikasi kasir Android yang:
- nyaman dipakai untuk transaksi cepat,
- dapat berjalan offline lebih dulu,
- mudah dipelihara,
- dan siap tumbuh ke standar produksi.

## Stack Utama
- Kotlin 2.x
- Java 17+
- Android Gradle Plugin 9.x
- Gradle 9.x
- Jetpack Compose
- Material 3

## Status Saat Ini
Level 0 - Fondasi proyek:
- setup Android Studio selesai
- proyek berhasil build dan run
- repository dan disiplin kerja sedang dirapikan

## Cara Menjalankan
1. Buka proyek di Android Studio.
2. Pastikan SDK dan emulator tersedia.
3. Jalankan aplikasi lewat build variant `debug`.

Atau lewat terminal:

### Windows
```bash
gradlew.bat assembleDebug
